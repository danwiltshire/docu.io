# Prerequisites

## Node.JS
```bash
curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
sudo apt-get install -y nodejs
```

## Express Generator
```bash
sudo npm install express-generator -g
```

# Generate code
```bash
express --view=pug docu.io
npm install
npx create-react-app docu.io
```
